# Business Problem

Financial institutions generate large volumes of customer, transaction, and marketing campaign data every day. However, extracting meaningful business insights from this data is often challenging due to fragmented systems and lack of centralized analytics.

The objective of this project is to design and develop an end-to-end Banking Customer Analytics System that helps stakeholders:

- Monitor customer churn trends
- Analyze customer transaction behavior
- Measure marketing campaign effectiveness
- Identify high-value customers
- Track business KPIs using dashboards

This project simulates a real-world banking analytics workflow using Python, PostgreSQL, SQL, and Power BI.

The system includes:
- Data generation
- ETL pipeline
- Relational database modeling
- SQL analytics
- Business intelligence dashboards

The final dashboards provide actionable insights for executive, customer, and marketing analytics teams.
